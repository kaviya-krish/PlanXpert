# PlanXpert - Travel Planner Web App 🌍✈️

## Project Overview
**PlanXpert Travel Planner** is a visually engaging travel planning web app that allows users to plan, track, and manage their trips effortlessly. Users can add destinations, calculate budgets, create day-wise itineraries, and keep track of expenses — all in one intuitive interface.

This project is built using **HTML, CSS, and JavaScript**, with a fully responsive design and interactive features.

---

## Project Goals
- Plan trips visually with an interactive interface  
- Track destinations, arrival/departure dates, vibes, and notes  
- Manage budgets with planned vs actual expenses  
- Visualize travel itineraries day-wise  
- Provide an engaging, responsive, and user-friendly experience  

---

## Tools & Technologies Used
- **HTML5** – Structure & markup  
- **CSS3** – Styling & responsive design  
- **JavaScript** – Interactivity & dynamic updates  
- **Local Storage** – Temporary data storage (for destination & expense info)
- 
---

## Features
### Destinations Tab
- Add destinations with city, dates, vibes (Beach, Mountain, Culture, etc.), and notes  
- Edit or remove destinations easily  
- Auto-calculate total days and display stats  

### Budget Tab
- Track planned vs actual expenses  
- Visualize spending by category using charts  
- Calculate running balance with color-coded indicators  

### Itinerary Tab
- Auto-generate day-wise itinerary from destination dates  
- Add activities with time, type emoji, duration, and notes  
- Add extra days manually if needed
